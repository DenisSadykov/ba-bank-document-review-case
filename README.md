# Тестовое задание для Бизнес-аналитика

Репозиторий синхронизирован с исходным Google Doc, который был передан как основной источник.

## Содержимое

- `outputs/ba_solution.pdf` — PDF-версия документа из Google Docs
- `outputs/ba_solution.txt` — текстовая выгрузка содержимого документа

## Примечание

- В репозитории оставлены только материалы, собранные из указанного пользователем источника.
- Временные и рабочие файлы исключены из версии.
