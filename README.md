# BA Test Assignment: Bank Document Review

This repository contains the prepared deliverables for a business analyst test assignment related to the primary review of a document package before submitting an advance payment request to a bank.

## Contents

- `outputs/ba_solution.pdf` — final written solution
- `outputs/ba_solution.txt` — editable text version
- `outputs/process_scheme.drawio` — editable process diagram for diagrams.net
- `outputs/process_scheme.svg` — vector export of the process diagram
- `outputs/process_scheme.svg.png` — PNG preview of the diagram
- `outputs/diagram_work/build_drawio_xml.py` — helper script used to build the draw.io XML structure
- `outputs/diagram_work/process_scheme_mxgraph.xml` — generated draw.io XML model

## Notes

- Temporary working files are excluded from version control.
- Source materials downloaded during the task are not included in the public repository.
